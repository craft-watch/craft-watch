## Frontend

Stuff to install:

```
brew install yarn
```

Then in `frontend/`:

```
yarn start
```

Then browse to http://localhost:3000.


## Infra

Stuff to install:

```
brew install terraform
brew cask install google-cloud-sdk
```

Then set up GCloud SDK:

```
gcloud init
gcloud auth application-default login
```

Ensure you have the **Storage Admin** role.
